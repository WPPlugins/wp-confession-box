Wp Confession Box plugin allows visitors to do confession , read other confessions without showing their identity to others. 

== Description ==

This plugin contains following features - 

1. This plugin contains a jquery based submission form with validations .

2. A confession window which displays confessions.

3. Everything works using ajax and jquery.

4. New feeds pop box at top of confession window to show new feeds once click on it .

5. Allows site admin to block and delete any confession of window.

6. Fully responsive with design with simple user interface .

7. Shortcode [wp-confession-box] displays user interface .



== Installation ==

This section describes how to install the plugin and get it working.

Guide -

1. Extract the zip file.
1. Upload `wp-confession-box` directory to the `/wp-content/plugins/` directory .
1. Activate the plugin by the 'Plugins' menu from wp-admin area .
1. Use shortcode '[wp-confession-box]' inside wp-admin pages to show user interface and for php files use <?php echo do_shortcode('[wp-confession-box]'); ?>


== Frequently Asked Questions ==

= How to submit a confession ? =

Simply click on 'view confession form' button just below confession window and submit the form simple. 

= Does this form contains validations =

Yes, there are simple ajax validations with form .

= How to show confession form and confession window in my blog page ?  =

Use shortcode [wp-confession-box] inside pages.

= Does this plugin supports multilingual? =

No, Current version doesn't for now.
